# Marp-ucas-simple
一个简单的Marp for UCAS示例，基于[Awesome-Marp](https://github.com/favourhong/Awesome-Marp)进行简单的修改，主题色修改为学术蓝。
其中用到的国科大图标均在中科院官网下载。
